# 标记型歌词 {#annotaion-lyrics}

标记型歌词是一种类似标记符号的语法，用于在没有音符的位置插入歌词。这样的歌词在音乐剧乐谱等场景中较为常见，一般用来表示没有固定节奏的台词或念白。

## 标记型歌词语法 {#syntax}

标记型歌词行是依附于一般歌词行的结构，与标记符号行格式是相同的，但是以 `La:` 开头，添加在歌词行对应的 `L` `Lc` 或 `Lw` 下方。

在 28-30 三个小节中，「I can't wait to meet everyone! Oh! What if I meet, the one?」一段歌词属于念白内容，上方没有任何音符，表明这些念白的节奏并不重要。

标记型歌词行的「音符」只能是文本标记，而不能是力度或和弦标记。
