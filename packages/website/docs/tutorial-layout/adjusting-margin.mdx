# 行间距 {#adjust-margin}

默认情况下，为了保证良好的可读性并且容纳各类记号，Sparks NMN 的排版是多少有些「浪费纸」的。因此，你可以通过一些选项来调整行间距，使得乐谱变得更加紧凑。

## 所有间距选项 {#all-options}

<Cr.RenderPropsReference filters={['margin']} />

- 所有间距及负间距的单位均为 `em`，表示页面宽度的 $\frac{1}{122}$。
