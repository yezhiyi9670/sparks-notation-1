# 换气记号、小括号、层叠连缀 {#insertation}

插入记号可以理解成一种能插入在两个音符中间的特殊音符，不会占据任何时值或排版空间。

## 所有插入记号 {#all-insertations}

<Cr.InsertationReference />

- 在仅记载主旋律的乐谱中，间奏和人声谱面会写在同一声部。有的人习惯将间奏用括号括起来，以示区分。这就是括号插入记号的用法。
- 层叠连缀用在两个音中间，表示后面的音开始时，前面的音实际还没有结束（实质上构成多声部关系）。
