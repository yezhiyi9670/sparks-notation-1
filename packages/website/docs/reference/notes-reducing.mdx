# 减时线、三连音、多连音 {#notes-reducing}

## 减时线

减时线即音符的下划线，用于将音符的时长减半。

减时线使用小括号表示，需要减半的一个或多个音符放到小括号内，这样可以同时表达分组的信息。

<C.ScoreExample canLoad children={
`P: 1=? 4/4
---
// highlight-next-line
N: ((6d135)) (3(32)) (3(32)) (3(6d7d)) | ((1321)) (6d5d) 3d - |((6d135)) (3(32)) (3(32)) (3(6d7d)) | ((1321)) (6d5d) 6d - |`
} />

:::caution 常见错误

任何情况下，你不能给一个环境中的音符在另一个环境内添加延时线。这样尽管看起来不会出错，但实际上会导致解析器误解。  
另外，不应该在减时线内使用延时线。

:::

## 三连音

八分三连音或更短的三连音通过在减时线括号前添加字母 `T` 实现。

<C.ScoreExample canLoad children={
`P: 1=? 4/4
---
// highlight-next-line
N: 1e (01e) 7 (07) | 6 T(671e) 7 5 | 1e (01e) 7 (07) | 6 T(671e) 7 5 |
---
// highlight-next-line
N: 4 (04) 1e (01e) | 3 T(345) 4 3 | 4 (04) 1e (01e) | 3 T(345) 4 3 |
`
} />

[三等分法则](/docs/reference/musical-props#beats)启用时，八分音符会自动变成八分三连音（即十二分音符），不需要添加字母 `T`。

四分三连音通过在字母 `T` 后跟三个音符实现。

<C.ScoreExample canLoad children={
`P: 1=? 4/4
---
// highlight-next-line
N: 6. (5) T4 3 4 | (54) (32) 3 - | 4. (3) T2 3 4 | 5 - - 0 |
`
} />

:::caution 避免滥用

如果记谱的时候遇到了什么需要用四分三连音表示的东西，请先仔细想想它是不是 3-3-2 节奏型。如果是，则不应该用三连音表示。

:::

## 多连音 {#n-plet}

多连音是三连音的一般化，指的是 3 以上的连音。三连音记号的要点是这些：

- 三连音记号下，音符长度变为原来的 $\frac{2}{3}$。
- 对于四分三连音，字符 `T` 作用于其后的 $3$ 个音符。
- 三连音记号上标的数字是 $3$。

推广到多连音的情况：

- 多连音记号下，音符长度变为原来的 $\frac{x}{y}$。
- 对于四分多连音，记号 `T[x,y]` 作用于其后的 $y$ 个音符。
- 当作用于一条减时线，或者满足 $y=x+1$ 时，三连音记号上标的数字是 $y$，否则三连音记号上标记 $y/x$。

多连音的记号就是 `T[x,y]`。多连音组所选择的减时线层数应当使得 $\frac{x}{y}$ 尽可能接近 $1$。

<C.ScoreExample canLoad children={
`P: 1=C 4/4
====
// highlight-next-line
N: T[8,7]((3234543)) 2 4 |  // 7 连音，作用于减时线，共占据 2 拍（减时线的层数应使得 x/y 的值尽可能接近 1）
---
// highlight-next-line
N: T[3,5]1 2 3 2 1 7d | // 5 连音，作用于 5 个四分音符，占据 3 拍
---
// highlight-next-line
N: T[6,5](12321) 7d | // 上一个小节的另一种写法。由于 6/5 更接近 1，这样写更加合适。
---
// highlight-next-line
N: T[4,5]6d 7d 1 7d 1 ||| // 5 连音，作用于 5 个四分音符，占据 4 拍
`
} />

算清楚 $x$ 的值，需要对时值有较好的理解，并有一定的理性思维。不过，既然你已经在接触多连音了，这肯定不是个难事，对吧？

## 三等分法则下使用八分音符

如果在三等分法则下需要让四分音符减半而非减为 $\frac{1}{3}$，在减时线括号前添加字母 `D` 即可。

<C.ScoreExample canLoad children={
`P: 1=bE 6/4T
Rp: n=3
---
// highlight-next-line
N: (1~12) (3~34) (5~53) 2 5 2 | 4 D(43) D(27d) 1 - - |
`
} />
